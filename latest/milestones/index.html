    🙏 给未来维护者的一封信¶

    如果你正在读这段文字，说明你正在接手一个有人认真打磨过的项目。

    这个项目的代码不复杂 —— 一个 FastAPI 入口 + 5 个中间件 + 1 个本地工具集； 但每个决策都权衡过：哪些必须做、哪些不必要、哪些先 stub 后续再补。

    请保持这些原则：

    1. 隐私优先 —— 永远不要让用户的 API Key 离开他们的机器
    2. 测试驱动 —— 任何中间件改动都加新测试
    3. 文档先行 —— 用户痛点写进 README 的 FAQ
    4. 小步迭代 —— 一个 PR 解决一个明确问题

    谢谢你读到这里。

    — 上一个维护者
